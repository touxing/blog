# 如何写一个cli工具并发布npm包

  1. 注册 npm 账号
  2. 本地初始化一个 npm 项目 npm init -y
  3. 本地测试 npm link 把cli命令链接到环境变量中
  4. 本地执行 cli xx 命令，有问题，继续改；没问题，增加 README.md 项目说明文件，准备提交
  5. npm login 登录 npm 仓库
  6. npm publish 提交到 npm 仓库
